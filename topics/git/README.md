### Что такое git?

Git - это широко используемая распределённая система контроля версий с открытым исходным кодом. Она позволяет легко разрабатывать как небольшие, так и крупные проекты.

Над одним проектом могут одновременно работать несколько разработчиков, которые могут отслеживать внесенные и отраженные в коде изменения. Кроме того, если что-то пойдет не так, вы сможете без труда вернуть изменения к предыдущей версии.

<summary>1. Как исправить последний коммит ( amend ) ✏️</summary>

Сделали очередные изменения? 
А оно относится к предыдущему коммиту, не хочешь много коммитов

#### Шаги:
```bash

    # Добавляем изменения в staging**
    git add хз.txt

    # С новым сообщением
    git commit --amend -m "Это хз"

    # Или без изменения сообщения (самый удобный вариант)
    git commit --amend --no-edit

    # Запушьте изменения (если коммит уже был отправлен)
    git push --force origin main
```

<summary>2. Скрыть часть репы для себя ✏️</summary>

Например, у меня есть папка .temp в которой я держу файлы с мыслями или другой личной информацией.
И хочу потом на другом компьютере подтянуть эту папку, те она должна быть общедоступной, но скрытой и подтягивается только для моего аккаунта.

Можно сделать через шифрование, а можно через подмодули?

Подмодули значит, что есть 2-ой приватный репозиторий, как подмодуль основного.

Рассмотрю через шифрование.

#### Шаги:
```bash

    # Удаляем всю папку из индекса(прекращаем отслеживать)
    git rm -r --cached .temp

    # С новым сообщением
    git commit --amend -m "Это хз"

    # Или без изменения сообщения (самый удобный вариант)
    git commit --amend --no-edit

    # Запушьте изменения (если коммит уже был отправлен)
    git push --force origin main
```
